--- BROCK BREAKING ---

Move the paddle using the arrow keys and spacebar to bounce the ball and break the blocks. The game is cleared when all blocks are destroyed. If the ball touches the bottom edge, the game is over.

矢印キーとスペースキーでパドルを動かし、ボールを跳ね返らせブロックを壊す。すべてのブロックを壊したらゲームクリア。ボールが下辺に触れたらゲームオーバー。

Javascriptを使用したブロック崩しゲームのチュートリアルによって作成されました。
